
cd: Dirigirse a otro directorio  
mkdir: Crear nuevo directorio  
touch : Crear nuevo archivo  
vim: Abrir editor de texto  
cat: Previsualizar archivo
git init: Crear un nuevo repositorio en una carpeta  
git status: Conocer cuales archivos no estan en el repositorio y si hay algun archivo en el stage  
git add: Agregar un nuevo archivo al stage  
git commit -m "": Crear un nuevo commit  
git push: Subir el repositorio a la nube  
git log: Revisar el historial de repositorios
